# 微信公众号排版工具

## Overview

将 Markdown 文章转换为微信公众号可直接粘贴的简洁 HTML 格式。

**设计理念**：
- **保留原文**：不添加任何额外标签，不修改文字内容
- **简洁排版**：清晰的层级，适度的留白
- **重点突出**：仅对真正重要的内容加粗

## When to Use

当用户需要将本地 Markdown 内容排版并发布到微信公众号时使用：

- 用户说"帮我排版公众号文章"
- 用户说"这个 md 要发到公众号"
- 用户需要干净、简洁的公众号排版

## Core Principles

| 原则 | 说明 |
|------|------|
| md要手动修改 | 基于原始md内容，手动修改新的排版的md |
| 简洁排版 | 清晰的标题层级，干净的段落 |
| 适度强调 | 只对真正的重点加粗 |

## Style Guide

### 视觉特征

- **纯净背景**：纯白背景，无彩色渐变
- **无复杂装饰**：去阴影、去渐变
- **清晰层级**：标题有明确的视觉区分
- **克制强调**：加粗仅用于真正的重点

### 配色方案

| 用途 | 颜色值 |
|------|--------|
| 主题色 | `#2563eb`（蓝，用于链接、加粗） |
| 正文色 | `#374151`（深灰） |
| 辅助色 | `#6b7280` |
| 分割线 | `#e5e7eb` |

### 标题样式

| 层级 | 字号 | 样式 |
|------|------|------|
| 一级 `#` | 24px | 居中，黑体，底部细线 |
| 二级 `##` | 20px | 居中，加粗 |
| 三级 `###` | 17px | 左对齐，左侧竖线 |

### 段落样式

- 字号：16px
- 行高：1.8
- 段间距：16px

### 重点强调

- **加粗**：`color: #2563eb`（主题色）
- **引用**（`>`）：左侧细边框 + 浅灰背景

## Usage

### 命令

```bash
python scripts/formatter.py article.md
```

### 输出说明

转换后的 HTML 代码直接粘贴到微信公众号编辑器（点击「源码」按钮粘贴即可）。

## Workflow

### 特别注意step 2

### Step 1: 读取原始 Markdown

读取用户提供的 `.md` 文件，保留原始内容。

### Step 2: 优化 Markdown 格式

执行以下优化（不修改文字内容）：

1. **添加标题层级**：根据内容逻辑添加 `##`、`###`
2. **保留原文**：不添加任何类型标记
3. **突出重点**：对关键数据、核心观点加粗 `**`
4. **章节分隔**：用 `---` 分隔主要章节

### Step 3: 执行 HTML 转换

```bash
python .claude/skills/wechat-public-account-formatter/scripts/formatter.py <input.md> --save-html
```

### Step 4: 提供结果

将 HTML 代码提供给用户，附上使用说明。
